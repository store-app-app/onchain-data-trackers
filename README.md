### 1. 实体去匿名化与地址追踪

#### [Arkham Intelligence](https://www.usdtbi.com/analytics/arkham) 
**深度解析**：Arkham 是一款革命性的去中心化智能分析平台，专注于“去匿名化”区块链交易。它利用强大的人工智能算法，将匿名的加密钱包地址与现实世界的实体（如交易所、风推基金、名人巨鲸）进行匹配绑定。其内置的追踪图谱功能，是链上资金流向分析和黑客事件溯源的顶级侦察利器。
**核心优势与场景**：
| 核心优势 | 适用场景 |
| :--- | :--- |
| 实体标签识别系统 | 监控巨鲸地址动向与 Smart Money 资金流 |
| 资金流向拓扑图 | 黑客攻击追踪、洗钱分析及项目方跑路预警 |

### 2. 高级可视化与数据多维看板

#### [Dune Analytics](https://www.usdtbi.com/analytics/dune)
**深度解析**：Dune Analytics 是 Web3 数据分析师和研究员的必备开源平台。它通过将复杂的区块链底层原始数据转化为易于查询的结构化 SQL 数据库，让用户能够直观地创建可视化数据看板（Dashboards）。无论是追踪复杂 DeFi 协议的 TVL 变化，还是分析 NFT 市场的每日换手热度，Dune 都提供了极高的透明度。
**核心优势与场景**：
| 核心优势 | 适用场景 |
| :--- | :--- |
| 原生 SQL 查询支持 | 投研机构进行宏观行业数据分析与图表生成 |
| 庞大开源看板社区 | 散户挖掘宏观趋势与监控特定协议健康度 |

#### [Nansen](https://www.usdtbi.com/analytics/nansen)
**深度解析**：Nansen 是专门服务于机构与专业交易者的链上数据分析天花板。它通过对数亿个钱包地址进行精细化的标签归类（如 Smart Money, Legendary NFT Hodler, Flash Boy），帮助用户在庞杂的噪音中过滤出真正有价值的真实动向，是捕捉链上 alpha 信号、洞察机构早期建仓行为的王牌工具。
**核心优势与场景**：
| 核心优势 | 适用场景 |
| :--- | :--- |
| 标签精细化清洗机制 | 探测机构及 Smart Money 的早期埋伏币种 |
| 代币持有者轮廓分析 | 监控核心大户在特定代币暴跌前的出货行为 |

* [[DexCheck]](https://www.usdtbi.com/tools/dexcheck) - 深度整合 AI 算法的 DEX 交易分析工具，内置 Smart Money 追踪器与 Meme 币狙击雷达。
* [[Bubblemaps]](https://www.usdtbi.com/analytics/bubblemaps) - 独特的泡泡图可视化代币供应分布工具，一眼识破项目方老鼠仓与操盘手关联地址。

### 2. 代币经济学与解锁监控 (Tokenomics & Vesting Trackers)
* [[Token Unlocks]](https://www.usdtbi.com/tokenomics/tokenunlocks) - 全网最专业的代币解锁日程追踪看板，精准监控各大 VC 及团队份额的释放高危期。
* [[DropsTab]](https://www.usdtbi.com/tokenomics/dropstab) - 综合性的加密资产投研门户，提供极其精细的轮次融资价格、估值以及解锁比例计算。
* [[Vestlab]](https://www.usdtbi.com/tokenomics/vestlab) - 专注于早期项目代币分发、锁仓（Vesting）图表与IDO白名单周期的极简监控台。

* [[Whale Alert]](https://www.usdtbi.com/alerts/whalealert) - 全球使用最广的独立大额交易监控服务，秒级推送 CEX 与钱包之间的数百万美元级转账。
* [[Lookonchain]](https://www.usdtbi.com/news/lookonchain) - 知名链上爆料推手的数据工具，专注于提炼并捕获巨鲸与 Smart Money 在链上暴赚的密码。
